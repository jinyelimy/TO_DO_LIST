# 옐링그의 To-do List

React와 Vite로 만든 감성형 투두 대시보드입니다.  
한 화면에서 날짜 선택, 할 일 관리, 진행률, 날씨, 명언, 통계를 함께 볼 수 있도록 구성했습니다.

## 주요 기능

- 날짜별 할 일 관리
  - 캘린더에서 날짜를 선택하면 해당 날짜의 할 일, 진행률, 통계가 함께 갱신됩니다.
  - 월 이동 화살표로 이전달/다음달을 확인할 수 있습니다.
  - 캘린더는 바깥 달 날짜도 회색으로 함께 보여줍니다.

- 할 일 추가 / 수정 / 삭제
  - 모달에서 새 할 일을 추가할 수 있습니다.
  - 기존 할 일 수정 기능이 포함되어 있습니다.
  - 시작 시간, 종료 시간을 시간 선택 드롭다운으로 지정할 수 있습니다.
  - 종료 시간이 시작 시간보다 빠르면 경고가 표시되고 저장되지 않습니다.

- 정렬 및 필터
  - 할 일은 시작 시간 순으로 정렬됩니다.
  - 시간이 같으면 `일정 -> 일상 -> 업무 -> 자기개발` 순으로 정렬됩니다.
  - 카테고리 필터를 지원합니다.
  - 완료 항목이 있을 때만 완료 항목 숨기기 버튼이 표시됩니다.

- 날짜 제한 처리
  - 오늘보다 지난 날짜는 할 일 추가 / 수정 / 삭제가 제한됩니다.

- 진행률 카드
  - 완료율에 따라 말풍선 문구가 달라집니다.
  - 진행률 러너 UI와 퍼센트 정보를 함께 보여줍니다.

- 날씨 카드
  - 서울 기준 실시간 날씨를 표시합니다.
  - 현재 기온, 체감 온도, 습도, 하늘 상태를 함께 보여줍니다.

- 오늘의 한마디
  - 내부 명언 데이터셋을 사용합니다.
  - 버튼으로 다른 명언을 바로 바꿔볼 수 있습니다.

- 데이터 유지
  - 로컬 스토리지 기반으로 데이터를 저장해 새로고침 후에도 유지됩니다.

## UI/UX 포인트

- 노란 톤 중심의 감성형 대시보드
- 데스크톱 중심 레이아웃
- 해상도별 반응형 보정
- 카드형 정보 구조
- 한국어 중심 인터페이스

## 기본 데이터

- 샘플 할 일이 일부 포함되어 있어 초기 상태에서도 레이아웃과 날짜 흐름을 확인할 수 있습니다.
- `2026-03-09` 날짜에는 캘린더 연동 확인용 예시 데이터가 포함되어 있습니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://127.0.0.1:5173`으로 접속하면 됩니다.

## 빌드

```bash
npm run build
```

## 참고 사항

- 날씨 정보는 `Open-Meteo API` 응답에 따라 달라질 수 있습니다.
- 현재 프로젝트는 별도 테스트 러너 없이 `vite build` 기준으로 검증하고 있습니다.
